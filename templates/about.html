A precise type of formal grammar is used to describe the mecanisms involved in plants growth. Grammars of this type are called L-systems. L-Py is a simulation software and a powerful language that mixes L-systems with the Python high-level modeling language. The software allows users to create their L-Py programs and to display the results thanks to the integrated PlantGL 3D Viewer.

L-PyWeb allows users to make their simulations online without installing any package. It uses the same primitives and keywords than the software version.
